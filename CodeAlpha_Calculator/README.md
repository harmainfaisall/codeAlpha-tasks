🧮 Calculator App

An interactive and responsive calculator built using HTML, CSS, and JavaScript.
This calculator performs basic arithmetic operations and includes button touch sound effects to create a more engaging and interactive user experience.

✨ Features

Responsive calculator design 📱💻

Perform basic arithmetic operations ➕➖✖️➗

Button click/touch sound effects 🔊

Clear and delete functionality 🧹

Smooth button animations ✨

Clean and modern UI 🎨

🛠️ Technologies Used

HTML5

CSS3

JavaScript

📂 Functionalities

Addition

Subtraction

Multiplication

Division

Clear screen

Delete last input

Interactive sound feedback

📌 Purpose

This project was created to practice JavaScript logic-building and improve frontend development skills by creating a functional and interactive calculator application.
